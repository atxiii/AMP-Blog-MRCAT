This plugin convert your blog post to Google AMP.

== Description ==
#AMP is a web component framework to easily create user-first  websites. This Plugin convert your blog post and other posts to Google AMP.


== Installation ==

1. Upload `amp-mrcat.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
